# Bootstrap Gulp Sass 
# Starter kit
### Technical Requirements

Install with npm: 

###  Development Dependencies
    -  gulp: 3.9.1,
    -  gulp-clean-css: 3.9.0,
    -  gulp-concat: 2.6.1,
    -  gulp-minify: 1.0.0,
    -  gulp-sass: 3.1.0,
   

###  Dev Dependencies
    - font-awesome: 4.7.0,
    - popper.js : 1.14.3

###  Production Dependencies 
    - jQuery v.3.3.0,
    - bootstrap: 4.1.0,



### Development environment

- Clone repository.
- Positioned in our project run npm install to download and install all the dependencies used.
- run  $ gulp
